# سورین صنعت فرزان | surinsf.com

وبسایت رسمی شرکت سورین صنعت فرزان - واردات و صادرات تجهیزات برق صنعتی، اتوماسیون و ابزار دقیق

## تکنولوژی
- **Next.js 15** با App Router و static export
- **React 19** + TypeScript
- **Tailwind CSS 3**
- **GSAP** برای انیمیشن‌های اسکرول
- **Three.js** برای پس‌زمینه سه‌بعدی

## شروع سریع
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
خروجی استاتیک در دایرکتوری `out/`

## Deploy
مستقر روی **Cloudflare Pages** با دامنه https://surinsf.com
